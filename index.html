Comandos GIT:

Git status: mostra o status da árvore de trabalho;
Git push: atualiza referências remotas junto com objetos associados;
Git pull: busca e integra com outro repositório ou uma ramificação local;
Git commit: grava alterações no repositório;
Git add: adiciona o conteúdo do arquivo ao índice.

O que é o GIT

Git é um sistema de controle de versão distribuído gratuito e de código aberto projetado para lidar com tudo, desde projetos pequenos a muito grandes com velocidade e eficiência.

Como utilizar o git em casa, para aprender e colocar em prática? Utilize um editor a sua escolha e crie uma conta no GitHub para subir seus projetos.
